# Minecraft-Join-Message-Plugin-1.0
Minecraft Join Message Plugin 1.0

-------------------------------------------
# GENERAL INFORMATION
Versions You Can Use the Plugin: **1.21.4**


-------------------------------------------

# COMMANDS
**/setmsg [Message]**

*When you use this command, it will automatically become your login message. Every time you enter the game, it will be written as NAME [Message]*
